# Sessão 02+03 — Pré-requisitos

## Objetivo
Executar sessão consolidada de Airflow + CDC + formatos/tipos de tabela.

## Dependências
- Sessão 02 (Airflow 3.1.x) disponível.
- Sessão consolidada 02+03 disponível.

- 12 GB RAM livres.
- 4 vCPUs recomendados.

## Ambientes suportados
- Ubuntu
- Windows + Docker Desktop + WSL (executar comandos no terminal WSL)
