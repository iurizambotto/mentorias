# Sessão 06 — Pré-requisitos

## Objetivo
Executar ingestão com Airbyte (`abctl`) e destinos locais (MinIO + PostgreSQL).

## Decisão técnica
- Airbyte será operado via `abctl` por aderência ao quickstart atual.
- Docker Compose da sessão mantém dependências locais do laboratório.

## Recursos recomendados
- 16 GB RAM para rodar Airbyte + dependências.
- 6 vCPUs recomendados.
