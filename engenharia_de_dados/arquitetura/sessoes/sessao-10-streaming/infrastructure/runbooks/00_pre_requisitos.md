# Sessão 10 — Pré-requisitos

## Objetivo
Executar cenário completo da plataforma de dados (ingestão até governança e observabilidade).

## Requisitos recomendados
- 24–32 GB RAM disponíveis.
- 8 vCPUs recomendados.
- Docker Compose v2 atualizado.

## Observação
Antes de subir tudo, garantir que sessões isoladas não estejam rodando em paralelo para evitar sobrecarga.
