# zambotto-mentoria

## Visão geral
Projeto only-docs para organizar sessões, materiais e exercícios das mentorias de engenharia de dados conduzidas por Iuri Zambotto em parceria com Paulo Shindi. O mentorado inicial é Aurelius Oliveira.

## Estrutura
```
zambotto-mentoria/
├── README.md
├── AGENTS.md
├── TASKS.md
├── docs/
│   ├── notes/
│   ├── sessions/
│   ├── people/
│   ├── guides/
│   ├── decisions/
│   ├── prompts/
│   └── auto_generated/
└── logs/
    └── activity.md
```

## Fontes e referências
- `~/Downloads/Roteiro_Estudos_Aurelius.xlsx` (aba `Sessões`) — importado via pandas.
- `~/Downloads/Perfil (1).pdf` — arquivo não encontrado no ambiente; conteúdo textual fornecido na conversa.
- Texto do LinkedIn fornecido na conversa (PII autorizado).

## Como usar
- Planeje sessões em `docs/sessions/`.
- Registre decisões em `docs/decisions/`.
- Registre tarefas em `TASKS.md` e atividades em `logs/activity.md`.
- Centralize perfis dos mentorados em `docs/people/`.

## Política de PII
- PII só deve ser registrada com autorização explícita.
- Preferir resumos profissionais; dados de contato ficam em seção específica quando necessário.
